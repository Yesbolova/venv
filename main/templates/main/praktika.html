Python тілінің тарихы

Python тілін 1991 жылы голландиялық бағдарламашы Гвидо Ван Россум (Guido van Rossum) жасаған. Тілдің Python деп аталуының себебі Гвидо Британдық "Monty Python' s Flying Circus" комедиялық сериалының жанкүйері болған.

Windows операциялық жүйесіне Python тілін орнату

Python тілі еркін бағдарламалық жасақтама болып табылады, сондықтан оны Python тілінің ресми сайтынан жүктеп алуға, еркін таратуға және барлық заманауи операциялық жүйелерге орнатуға болады.

Python 3.11.2 программалау тілінің жұмыс ортасы. Практикум

1. Python 3.11.2 программалау тілінің жұмыс ортасы ашыңыз.

2. Программада жаңа терезе ашу үшін IDLE ашып, мәзірден File ⇒ New File таңдаңыз.

3. Бос терезе тақырыбы «Untitled» сөзімен ашылады. Жаңа терезеде мына кодты енгізіңдер: print ('Мен Python тілін оқуды бастадым’)

4. Мәзірден File ⇒ Save таңдаңыз немесе F5 пернесін басыңыз. Ашылған терезеде программа.py деп теріп, файлды жұмыс үстелі немесе құжаттар бумасына сақтаңыздар. Python-да құрылған программа кеңейтілімі .ру.
