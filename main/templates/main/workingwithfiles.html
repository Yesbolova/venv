Python-дағы файлдар

Python да файлдардың екі түрі бар:

  • Мәтіндік;
  • Бинарлық

    • Мәтіндік файлдар

    Бұл файлдардың мазмұны адамның оқуына түсінікті. Олар адам түсінетін таңбалар тізбегінен тұрады. Блокнот және басқа стандартты редакторлар файлдың осы түрін оқи және өңдей алады. Мәтін екі форматта сақталуы мүмкін: (.txt) - қарапайым мәтін және (.rtf) - "байытылған мәтін форматы".

    Бинарлық файлдар

    Екілік файлдарда деректер кодталған түрде көрсетіледі (қарапайым таңбалардың орнына тек нөлдер (0) және бірліктер (1) қолданылады). Көп жағдайда бұл тек бит тізбегі. Олар .bin форматында сақталады.

    Кез келген файлдық операцияларды үш үлкен кезеңге бөлуге болады:

  • Файлды ашу;
  • Операцияны орындау (жазу, оқу);
  • Файлды жабу

    • Файлды ашу - open() әдісі

    Python-да кірістірілген open () функциясы бар. Оның көмегімен компьютердегі кез-келген файлды ашуға болады. Техникалық тұрғыдан Python оның негізінде объект жасайды.

    Синтаксис келесідей:

    f = open(file_name, access_mode)
    • file_name = ашылатын файл атауы
    • access_mode = файлды ашу режимі. Ол мыналар болуы мүмкін: оқу, жазу және т.б. үшін, егер басқасы көрсетілмесе, әдепкі оқу режимі (r) болып табылады. Әрі қарай файлды ашу режимдерінің толық тізімі
    РежимСипаттамасы
    rТек оқу үшін.
    wТек жазу үшін. Егер көрсетілген атпен табылмаса, жаңа файл жасайды.
    rbТек оқу үшін (бинарлық).
    wbТек жазу үшін (бинарлық). Егер көрсетілген атпен табылмаса, жаңа файл жасайды.
    r+Оқу және жазу үшін.
    rb+Оқу және жазу үшін. (бинарлық).
    w+Оқу және жазу үшін. Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды.
    wb+Оқу және жазу үшін (бинарлық). Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды..
    aЖаңа мазмұнды қосу үшін ашады. Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды.
    a+Жаңа мазмұнды қосу үшін ашады. Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды.
    abЖаңа мазмұнды қосу үшін ашады. (бинарлық) Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды.
    ab+Жаңа мазмұнды қосу үшін ашады. (бинарлық) Егер көрсетілген атпен табылмаса, жазу үшін жаңа файл жасайды.

    Мысалы

    Мәтіндік файл жасайық example.txt және оны жұмыс үстеліне сақтаймыз.

    Оны ашу үшін келесі код қолданылады:

    Файлды жабу - Сlose() әдісі

    Файлды Python-да ашқаннан кейін оны жабу керек. Осылайша ресурстар босатылып, қоқыс тазартылады. Нысан басқа файлға тағайындалған кезде Python файлды автоматты түрде жабады.

    Файлды жабудың келесі жолдар бар:

    №1 әдіс

    Файлды ашқаннан кейін оны - Сlose() әдісі арқылы жабу оңай.

    Жабылғаннан кейін бұл файлды қайта ашқанға дейін пайдалану мүмкін болмайды.

    №2 әдіс

    Сондай-ақ, try/finally деп жазуға болады, ол файлды ашқаннан кейін онымен операциялар ерекше жағдайларға әкелсе, ол автоматты түрде жабылатынына кепілдік береді.

    Python-да файлдарды оқу және жазу

    Python-да файлдарды тиісті режимдер арқылы оқуға немесе оларға ақпаратты жазуға болады.

    read() функциясы

    read() функциясы файлды оқу режимінде (r) ашқаннан кейін оның мазмұнын оқу үшін қолданылады .

    Синтаксис

    file.read(size)

    Бұл жерде,

    • file =файлдың объектісі
    • size = оқылатын таңбалар саны. Егер олар көрсетілмесе, онда файл толығымен оқылады.

    Мысалы

    >>> f = open('example.txt','r')
>>> f.read(7)  #example.txt-тен 7 символды оқу 
'This is '

    readline() функциясы

    readline() функциясы файл мазмұнын қатарынан оқу үшін қолданылады. Ол үлкен файлдар үшін қолданылады. Оның көмегімен кез-келген уақытта кез-келген жолға қол жеткізуге болады.

    Мысалы

    test.txt файлын бірнеше жол арқылы жасаймыз:

    This is line1.
This is line2.
This is line3.

    test.txt файлында readline() функциясы қалай жұмыс жасайтынын қарастырайық.

    >>> x = open('test.txt','r')
>>> x.readline()  # бірінші жолды оқу
This is line1.
>>> x.readline(2)  # екінші жолды ашу
This is line2.
>>> x.readlines()  # барлық жолдарды оқу
['This is line1.','This is line2.','This is line3.']

    Соңғы жағдайда жолдар бір-бірінен қалай бөлінетініне назар аударыңыз.

    write() функциясы

    write() функциясы жазу режимінде ашылған Python файлдарына жазу үшін қолданылады.

    Егер сіз осы режимде жоқ файлды ашуға тырыссаңыз, онда жаңасы жасалады.

    Синтаксис

    file.write(string)

    Мысалы


    Құрылмаған xyz.txt файлын ұсынсақ. Ол жасалып, оқу режимінде ашылады.

    >>> f = open('xyz.txt','w')  #  жазу режимінде ашу
>>> f.write('Hello \n World')  #  файлға Hello World деп жазу
Hello
World
>>> f.close()  # файлды жабу

    Python-да файлдардың атауын өзгерту

    rename() функциясы

    rename() функциясы файлдардың атын Python-да өзгерту үшін қолданылады. Оны пайдалану үшін алдымен импорттау керек модуль os.

    Синтаксисі келесідей.

    import os
os.rename(src,dest)

    Бұл жерде,

    • src = атауын өзгерту керек файл
    • dest = файлдың жаңа атауы

    Мысалы

    >>> import os
>>> #xyz.txt файл атауын abc.txt деп өзгерту
>>> os.rename("xyz.txt","abc.txt")

    Python файлдарындағы ағымдағы позиция

    Python-да tell() функциясы арқылы файлдың ағымдағы позициясын білуге болады. Осылайша seek() командасы арқылы файлдың ағымдық позициясын өзгеруге болады.

    Пример

    >>> f = open('example.txt')  # example.txt, біз алдында жасаған файл
>>> f.read(4)  # алдымен 4-ші позицияға өтейік
This
>>> f.tell()  # ағымдағы позицияны қайтарады
4
>>> f.seek(0,0)  #  позицияны қайтан 0-ге қайтарады

    file.close()ашық файлды жабады
    file.fileno()файлдың бүтін дескрипторын қайтарады
    file.flush()ішкі буферді тазартады
    file.isatty()егер файл терминалға байланған болса, True мәнін қайтарады
    file.next()файлдың келесі жолын қайтарады
    file.read(n)файлдың алғашқы n таңбаларын оқу
    file.readline()жолдың немесе файлдың бір жолын оқиды
    file.readlines()файлдағы барлық жолдардың тізімін оқиды және қайтарады
    file.seek(offset[,whene])файлдағы ағымдық позицияны тағайындайды
    file.seekable()файл кездейсоқ қол жеткізуді қолдайтынын тексереді. True мәнін қайтарады.
    file.tell()файлдағы ағымдағы  позицияны қайтарады
    file.truncate(n)файл өлшемін азайтады. Егер N көрсетілсе, онда файл N байтқа дейін кесіледі, егер жоқ болса — ағымдағы позицияға дейін
    file.write(str)файлға str жолын қосады  
    file.writelines(sequence)файлға жолдар тізбегін қосады
