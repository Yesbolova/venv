1. Python тіліндегі if операторы

Python тіліндегі if операторының синтаксисі келесідей:

if шарт:
    # if операторының негізгі бөлігі

if операторы шартқа тәуелді.

  1. Егер шарт Ақиқат болса,онда if денесінің ішіндегі код оқылады.
  2. Егер шарт Жалған болса, онда if денесінің ішіндегі код өткізіліп жіберіледі.

2. Python тіліндегі if...else операторы

if операторында қосымша else ұсынылуы мүмкін.

if...else операторының синтаксисі келесідей: 

if шарт:
    # шарт ақиқат болғандағы код блогы

else:
    # шарт жалған болғандағы код блогы

The if...else statement evaluates the given condition:

Егер шарт Ақиқат мәнін қабылдаса,

  • if ішіндегі код орындалады.
  • else ішіндегі код өткізіліп жіберіледі.

Егер шарт Жалған мәнін қабылдаса,

  • else ішіндегі код орындалады,
  • if ішіндегі код өткізіліп жіберіледі.

3. Python тіліндегі if...elif...else операторлары

if...else операторы екі балама нұсқаның біреуі арқылы код блогын орындау үшін қолданылады.

Алайда, егер екіден көп балама арасында таңдау жасау керек болса, онда біз if...elif...else операторын қолданамыз

if...elif...else операторының синтаксисі келесідей:

if 1-шарт:
    # 1-код блогы

elif 2-шарт:
    # 2-код блогы

else:
    # 3-код блогы

Бұл жерде,

  1. Егер1-шарт Ақиқат мәнін қабылдаса, 1-код блогы орындалады.
  2. Егер 1-шарт Жалған мәнін қабылдаса, онда 2-шарт есептеледі.
    1. Егер 2-шарт Ақиқат болса, 2-код блогы орындалады.
    2. Егер 2-шарт Жалған болса, 3-код блогы орындалады.

Output

Жоғарыда келтірілген мысалда біз san деп аталатын және 0 мәніне ие айнымалы құрдық. Мұнда бізде екі шартты өрнек бар: Мұнда шарттың екеуі де Жалған деп бағаланады. Демек, else денесінің ішінде оператор орындалады.

Python тіліндегі кірістірілген if операторлары

Біз сондай-ақ, if операторының ішінде if операторын пайдалана аламыз. Бұл кірістірілген if операторы ретінде белгілі.

Кірістірілген if операторының синтаксисі:

# сыртқы оператор if
if 1-шарт:
    # оператор(лар)

    # ішкі оператор if
    if 2-шарт:
        # оператор(лар)

Ескертулер:

  • else және elif операторларын ішкі if операторына қоса аламыз.
  • ішкі if операторын сыртқы else немесе elif операторларының ішіне енгізуге болады.
  • if операторларының бірнеше деңгейін енгізе аламыз.
